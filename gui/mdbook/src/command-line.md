# Command Line
Festival has some command line flags, useful for adjusting playback.

Some examples:

Play the next song:
```bash
./festival --next
```

Seek backwards 25 seconds in the current song:
```bash
./festival --seek-backward 25
```

Skip 3 songs into the queue:
```bash
./festival --skip 3
```

Play the 10th song in the queue:
```bash
./festival --index 10
```
